live template

如果你用过 sout，psvm等快捷指令，那你一定对live template不会陌生

live template的配置在网上一搜就一大片，我这里想告诉你live template是支持Groovy，不知道Groovy？去看看这个，下面我以一个使用slf4j快捷生成某个方法的输入参数为例。

第一步

使用find action 寻找live Templates:

编写自定义模板（自行搜索），模板里面写：

private static final Logger LOG = LoggerFactory.getLogger($VAR1$.class);

配置$VAR1$，下拉框选择className：

现在就配置好了LOG的声明。

第二步

使用第一步的套路，不过这时候下面填：

LOG.info($params$, $params2$);

配置$params$, $params2$，代码分别是：

groovyScript("'\"' + _1.collect { it + ' is {}'}.join(',')+ '\"'",methodParameters())

groovyScript("'' + _1.join(' ,')+ ''",methodParameters())

现在就配置好了loginfo，我们来看一下效果：

当然，这只是我选择的一种格式，你可以继续改造，如你可以在此基础上添加方法名和类名，也可以使用groovy给某个类写一个全部属性 setter………
